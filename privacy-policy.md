Privacy Policy — Foldy Chrome Extension
Last updated: 2025

Foldy does not collect, transmit, or store any personal data on Foldy-owned servers.

SYNC STORAGE
Foldy uses chrome.storage.sync. Your data syncs across Chrome browsers signed in to the same Google account via Google's own infrastructure — no Foldy servers involved. If not signed in to Chrome, data is stored locally only.

PERMISSIONS
• tabs — to display open tabs so you can save them to folders
• bookmarks — to browse bookmarks inside Foldy; never transmitted externally
• storage — to save and sync folders via chrome.storage.sync
• system.memory (optional) — to display RAM usage; never stored or transmitted
• identity — to verify CWS purchase status for Premium; no personal data stored

DATA SHARING
Foldy does not sell, share, or transmit your data. Google handles sync under their own privacy policy.

FOLDER SHARING
The Share Folder feature downloads a JSON file locally. Foldy transmits nothing.

PREMIUM LICENSING
Purchases handled by Google via Chrome Web Store. Foldy only receives paid/unpaid status.
